Atividade Prática – Curso Front-End
Tema: Página de Login com Redirecionamento para Cyberbullying

Objetivo:

Desenvolver uma página web de login utilizando HTML, CSS e JavaScript, com usuários e senhas previamente definidos. Após o login correto, o usuário será redirecionado para uma página sobre Cyberbullying.

Desafio

Criar um sistema simples de login contendo:

Página 1 – Login

Elementos obrigatórios:

Campo Usuário
Campo Senha
Botão Entrar
Mensagem de erro caso login esteja incorreto
Usuários cadastrados:
Usuário	Senha:
Regras:

✅ Se usuário e senha estiverem corretos:
➡ Redirecionar para cyberbullying.html

❌ Se estiver errado:
➡ Mostrar mensagem: Usuário ou senha inválidos

Página 2 – Cyberbullying

Criar uma página informativa contendo:

Título:

Cyberbullying: Combata essa prática

Conteúdo mínimo:
O que é cyberbullying
Exemplos
Consequências
Como denunciar
Como prevenir
Extras (vale nota):

Imagem ilustrativa
Design responsivo
Tema escuro moderno
Botão sair e voltar ao login

Estrutura sugerida:
/projeto-login
 ├── index.html
 ├── style.css
 ├── script.js
 └── cyberbullying.html
